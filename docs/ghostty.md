# Ghostty

## Tabs

New tab: `ctrl shift t`
Goto tab 1: `alt 1`
Close this tab: `ctrl shift w`

## Splits

Goto split: `ctrl alt arrow`
New split down: `ctrl shift e`
New split right: `ctrl shift o`
Close split: `ctrl d` (only works with nothing written)
Toggle split zoom: `ctrl shift enter`

# Etc

Reload config: `ctrl shift ,`
Toggle fullscreen: `f11`
Toggle command palette: `ctrl shift p`
